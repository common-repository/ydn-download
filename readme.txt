== Description ==
The easiest way to <strong>download</strong> files via wordpress <strong>Download</strong> plugin.

Download types

* Link download type
* Image download type
* Button download type
* Subscription download type

usage of digital download manager short code

[ydn_download href="https://downloads.wordpress.org/plugin/ydn-download.zip" title="Donwload title"]

If you think that you have found a bug in <strong>Download</strong> plugin or have any questions, please feel free to contact us at edmon.parker@gmail.com.
